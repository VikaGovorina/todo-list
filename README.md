Todo lists app using Webpack
Одностраничное веб-приложение TODO листа для управления задачами и проектами с возможностью создавать, редактировать, удалять задачи и проекты, устанавливать приоритеты и дедлайны.


[Live Preview](https://vikagovorina.github.io/todo-list/)


JavaScript, HTML, CSS, Webpack, Web Storage API
